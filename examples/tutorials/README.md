# Tutorial Examples

These examples match the three onboarding tasks that are easiest to understand
without a full workflow dataset:

1. `01_build_tree.py`: build a tree and inspect its derived properties
2. `02_simulate_tree.py`: run a direct zero-D simulation through a tree
3. `03_apply_tree_bc.py`: attach a tree-derived boundary condition to a tiny
   one-outlet `svZeroD` config

The first and third examples are self-contained.

The second example requires `pysvzerod`.
